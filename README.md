# 💪 Mon Programme — PWA Musculation

Application web progressive (PWA) de suivi d'entraînement musculaire. Accessible sur Android et iOS, installable comme une app native, sans compte ni connexion.

## ✨ Fonctionnalités

- **Programme sur 4 jours** — Push / Pull / Jambes / Full Haut de corps
- **Suivi des séries** — Poids & reps pré-remplis depuis votre dernière séance
- **Chrono de repos** — Temps de repos automatique et adapté par exercice
- **Chrono de séance** — Démarrage manuel, durée enregistrée dans l'historique
- **Historique complet** — Suppression par séance, exercice ou série
- **Durée estimée** — Affichée sur chaque journée dans l'accueil
- **Multi-profils** — Données isolées par prénom, pas de mélange entre utilisateurs
- **100% offline** — Service Worker + cache, fonctionne sans connexion après le 1er chargement

## 📱 Installation

### Android (Chrome)
1. Ouvrir l'URL dans Chrome
2. Menu → *Ajouter à l'écran d'accueil*

### iOS (Safari)
1. Ouvrir l'URL dans Safari
2. Partager → *Sur l'écran d'accueil*

## 🗓 Programme

| Jour | Séance |
|------|--------|
| Lundi | Pecs & Triceps — poids libres, pluriarticulaire |
| Mardi | Dos & Biceps — tractions, rowing, tirages |
| Mercredi | Cardio (1 sur 2) |
| Jeudi | Jambes — Quadriceps & Ischios |
| Vendredi | Full Haut du Corps — épaules, pecs, dos, bras |

## 🛠 Stack technique

HTML · CSS · JavaScript vanilla · LocalStorage · Service Worker · PWA Manifest

## 🔒 Données

Toutes les données sont stockées **localement sur votre appareil** (localStorage). Aucune donnée n'est envoyée à un serveur.
